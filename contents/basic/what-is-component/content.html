组件

组件有很多种理解，前端实际工作中更喜欢把包含UI + 功能可复用（也不一定）的封装体认作组件。

最简单的组件

基本类型 

     // string component
        {
            const component ="hello,world"
            render(component,"body")
        }
   
     // html component
    {
        component = <hr />
        render(component,"body")
    }

     // int component 
        {
            const component =1000
            render(component,"body")
        }

     // expression component 
    {
        component = 1000
        //表达式
        render(<h1>表达式：{component},{1+1} ,{9 * 9}</h1>, 'body')
    }

函数组件

函数组件可以实现复杂逻辑

函数节点

     // function node  component 
    {
        render(() => {
            return <h1>这是一个函数节点</h1>
        }, "body")
    }

函数标签

     // function tag  component 
        {
            function AComponent() {
                return <h1>这是一个函数标签</h1>
            }
            render(<AComponent />, "body")
        }

Web Components

Web Components 

     // web components
    {
        define("a-tag", class extends Component {
            render() {
                return <h1>这是一个WebComponent</h1>
            }
        })
        render(<a-tag />, "body")
    }
